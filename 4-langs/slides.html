4 Types of Languages You Must Learn

1. Procedural/OO

Python, Go, C, C++, Java, Perl, PHP, Ruby, C#, rust, D, …

fact.py

def fact(n):
    total = 1
    for i in range(1, n+1):
        total *= i
    return total

fact.go

func fact(n int) int {
    total := 1
    for n > 0 {
        total *= n
        n--
    }
    return total
}

I’d Try C/Go

2. Assembly

Know the machine!

fact.s

_start:
    movl $5, %eax   # Send value to compute in eax
    jmp fact
end:
    movl %eax, %ebx # Set result as return value
    movl $1, %eax   # System exit
    int $0x80

fact:
    movl %eax, %ecx  # Loop n -1 times
    decl %ecx
fact_loop:
    imul %ecx, %eax  # n = n * n-1
    loop fact_loop
    jmp end

3. Functional

  • Lispish
    • Common Lisp - SBCL
    • Scheme - Racket
    • Clojure
  • Haskell - GHC
  • ML - OCaml

fact.scm

(define (fact n)
  (let loop ((n n) (acc 1))
    (if (zero? n)
      acc
      (loop (- n 1) (* n acc)))))

fact.py

def fact(n):
    def _fact(n, acc):
        if n == 1:
            return acc
        return _fact(n-1, acc*n)
    return _fact(n, 1)

https://xkcd.com/297/

fact.py

def fact(n):
    if n == 1:
        return n
    return n * fact(n - 1)

fact.hs

fact n = foldl (*) 1 [1..n]

fact.py

from operator import mul
from functools import reduce

def fact(n):
    return reduce(mul, range(1, n+1))

4. Logic

fact.pl

fact(1, 1).
fact(N, F) :-
    N > 0,
    N1 is N - 1,
    fact(N1, F1),
    F is N * F1.
?- fact(10, X).
X = 3628800 .

fact.py

def fact(n):
    if n == 1:
        return n
    n1 = n - 1
    return n * fact(n1)
